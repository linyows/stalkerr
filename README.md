![The Shining](http://goo.gl/7JPKQ)

Usage
-----

### Start Stalkerr

    $ gem install stalkerr
    $ stalkerr --help
    $ stalkerr -D

### Connecting to Stalkerr

    /join #github <username>:<password>
    /join #qiita <username>:<password>
